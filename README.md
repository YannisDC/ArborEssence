## Installation
Add the following to your Package.swift dependencies:
```swift
.package(url: "https://github.com/YannisDC/ArborEssence.git", from: "1.0.0")
```
Then add `ArborEssence` as a dependency for your target:
```swift
.target(
    name: "YourApp",
    dependencies: ["ArborEssence"]
)
```
